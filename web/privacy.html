Privacy Policy for Typst for PowerPoint

Last Updated: January 25, 2026

Typst for PowerPoint is a PowerPoint Add-in that allows users to create and insert equations using the Typst typesetting system directly within PowerPoint presentations. It is not affiliated with or endorsed by the Typst project, but instead a community-driven effort to bring Typst's capabilities to PowerPoint users.

The app is completely open source and MIT-licensed. We don't store any user data or personal information. Typst code and generated content stay local on your machine within your PowerPoint files. The only data that may be collected are standard web server access logs by GitHub Pages when loading the Add-in UI, which is hosted as a static site on GitHub Pages.

Overview

This Privacy Policy describes how Typst for PowerPoint ("the app", "our service", "we", or "us") handles information when you use our PowerPoint Add-in.

Information Collection and Use

We do not collect, store, or transmit any personal information or user data.

The Typst for PowerPoint Add-in operates entirely on your local machine (except for loading the UI at startup from a static web page containing HTML/JS/CSS). When you use the app to insert equations into your PowerPoint presentations:

  • All Typst code you write stays local on your device.
  • All generated images and content remain within your PowerPoint document.
  • No data is sent to our servers (we don't have any backend servers).
  • No analytics or tracking mechanisms are implemented in the app.
  • No user accounts are created or maintained.

Third-Party Services

The Typst for PowerPoint Add-in interface is hosted as a static website on GitHub Pages. When you load the Add-in within PowerPoint, GitHub Pages may collect standard web server access logs, which could include:

  • Your IP address
  • Browser type and version (of the WebView inside PowerPoint)
  • Date and time of access

These logs are collected and managed by GitHub, not by us. Please refer to GitHub's Privacy Statement for information on how GitHub handles this data.

Data Security

Since we do not collect or store any personal information, there is no user data that could be compromised. All content you create using the app remains within your PowerPoint files. There is no personal information for you to access, correct, or delete.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date. We encourage you to review this Privacy Policy periodically for any changes.

Contact Us

If you have any questions or concerns about this Privacy Policy or the Typst for PowerPoint Add-in, please contact us through our GitHub repository.
