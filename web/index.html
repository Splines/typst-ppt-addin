PPTypst | Bring Typst to PowerPoint

PPTypst is a PowerPoint Add-in that allows users to insert and edit equations using the Typst typesetting system directly within PowerPoint presentations. It has a real-time preview of the formula and you can even generate from a file. It is not affiliated with or endorsed by Typst, but instead a community-driven effort to bring Typst's capabilities to PowerPoint users.

See the project on GitHub. It is currently not yet available in the PowerPoint Add-in Marketplace. See the repository for early preview installation instructions.

The PowerPoint Add-in will connect to this page in order to display the Add-in UI. All data stays local to your machine. In fact, there is no backend at all as this static site is hosted on GitHub Pages. Please see our Privacy Policy for more details.
